API Usage

You can also use the API directly with curl:

curl -X POST http://localhost:5000/process \ -F "file=@/path/to/your/image.jpg" \ -F "num_colors=5" \ -F "use_custom_grid=true" \ -F "grid_size=8" \ -F "use_smoothing=true" \ -F "smoothing_sigma=1.5" \ -F "enhance_contrast=true" \ -H "Accept: application/json"
